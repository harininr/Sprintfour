FEATURE 1
AI PRIVACY AUDITOR
====================================================================

Create a new feature called

Privacy Auditor

The Privacy Auditor should be accessible from the Review page as a collapsible AI assistant panel and also from the top navigation.

This is NOT a generic chatbot.

It is a context-aware AI assistant that understands:

• Original document
• Current document
• AI detections
• Consensus results
• User corrections
• Review status
• Remaining PII
• Export readiness

The assistant should answer questions based ONLY on the current review session.

Example questions:

- Have all dates been redacted?
- Did we miss any phone numbers?
- Are there still any names visible?
- Show every remaining email.
- List all passport numbers.
- Which entities have low confidence?
- Why wasn't this address redacted?
- Can I safely upload this document to ChatGPT?
- Which AI models disagreed?
- Show all doubtful PII.
- Give me a summary of this review.
- Generate a compliance report.

The backend should gather all session information and provide it as context to the AI.

Never allow hallucinations.

The AI must answer using only available review data.

Create suggested prompts underneath the input box.

Examples

Check if all dates are redacted

Show remaining phone numbers

Can I safely share this document?

Explain AI disagreements

Summarize review

The panel should have

Modern chat UI

Typing animation

Streaming responses if possible

Professional cards

Lucide icons only

Beautiful typography

Scrollable conversation history

The conversation should persist while the review session is active.
